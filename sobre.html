📘 Sobre o Projeto

Este sistema web foi desenvolvido como projeto final da disciplina Programação para Web I.

Ele permite adicionar, concluir, reabrir e excluir tarefas, além de acompanhar o progresso com uma barra animada.

Funcionalidades extras incluem filtros de tarefas, frases motivacionais e exibição da data/hora atual.
